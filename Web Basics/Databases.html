Installing Apache and PHP - macOS Big Sur
Step 1: First press command space to seach and open your terminal.
Step 2: Apache should come preinstalled so to start the Apache server just type the following into your terminal, you may also need to type your computers password if you are prompted 

sudo apachectl start

Step 3: Next open a broswer of your choice and type localhost, you should see the words "It works!" appear on your screen
Step 4: Now type the code below into your terminal and uncomment these three lines: 
sudo nano /etc/apache2/httpd.conf


Step 5: Press control x followed by y and enter to save and exit
Step 6: Check your Apache version by typing the following into the terminal and write this down for later 

httpd -v

Step 7: Now type the following into your terminal replacing username with your computers name. To find this name go to settings> Users & Groups> click the lock> enter your computer password> right click on the current user> click advance options> find where it says "Home directory: /Users/username/" this is your username 
sudo nano "/etc/apache2/users/username.conf"

Step 8: Next type this into the file and then press control x, y and enter to save and quit:


Step 9: Now set the files permission settings by typing the following: 

sudo chmod 644 "/etc/apache2/users/username.conf"

Step 10: Now type the code below into your terminal 
sudo nano /etc/apache2/extra/httpd-userdir.conf
Step 11: Next uncomment this line: 
Include /private/etc/apache2/users/*.conf
Step 12: Press control x followed by y and enter to save and exit
Step 13: Then open the httpd.conf file in your terminal again: 
sudo nano /etc/apache2/httpd.conf
Step 14: Now press control w and search for all instances of the word "override" and change "None" to "All"
Step 15: Press control x followed by y and enter to save and exit
Step 16: Now type the following and press enter to restart your Apache server in your terminal
sudo apachectl restart
Step 17: Finally check that your sites folder exists in your Users/username/ directory
